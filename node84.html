Figure 4.4: Framework of WWW Search Engine

The Figure 4.4 is a framework of a WWW search engine using the persistent storage for the indexing and searching. The persistent storage is suitable for such application.

The explanation of the specifications of the search engine is too much for this chapter. The persistent storage is initially designed for the search engine. However, the persistent storage is so designed that almost all data structures and types possible are suitable for its contents. It has come up to be a general-purpose persistent storage system for the C++ language.
